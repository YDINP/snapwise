**깨진 유리창 이론(Broken Windows Theory)**: 작은 무질서를 방치하면 더 큰 범죄로 이어진다는 이론입니다.

1982년 사회학자 제임스 윌슨과 조지 켈링이 제안했습니다.

**핵심 논리:**
건물 유리창 하나가 깨져 있으면 → 관리되지 않는다는 신호 → 더 많은 유리창이 깨짐 → 무질서 확산 → 범죄 증가

**뉴욕 지하철 실험 (1990년대):**
낙서 제거, 무임승차 단속 강화 → 중범죄 75% 감소

**코드에도 적용:**
작은 기술 부채(주석 없는 코드, 테스트 누락)를 방치하면 전체 코드 품질이 급격히 하락합니다.

**실천법:** 작은 문제를 즉시 해결하는 문화를 만드세요. 환경이 행동을 만듭니다.
