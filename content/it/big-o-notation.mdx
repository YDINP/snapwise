알고리즘 성능을 나타내는 **시간 복잡도** 표기법입니다.

**빠름 → 느림 순서:**

- **O(1)** - 상수 시간: 배열 인덱스 접근
- **O(log n)** - 로그 시간: 이진 탐색
- **O(n)** - 선형 시간: 배열 순회
- **O(n log n)** - 선형 로그: 병합 정렬, 퀵 정렬
- **O(n²)** - 제곱 시간: 중첩 반복문
- **O(2ⁿ)** - 지수 시간: 피보나치 재귀

**실무 팁:** n이 1,000일 때 O(n²)는 1백만 번, O(n log n)은 1만 번 연산입니다. 알고리즘 선택이 성능을 좌우합니다.
