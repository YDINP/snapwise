**Grid는 2차원(행+열), Flexbox는 1차원(행 or 열)**입니다.

Grid는 페이지 전체 레이아웃에 적합하고, Flexbox는 컴포넌트 내부 정렬에 최적화되어 있습니다.

실무에서는 둘 다 함께 사용합니다. 예를 들어 Grid로 페이지를 나누고, 각 영역 내부는 Flexbox로 정렬하는 방식입니다.

**Grid 사용**: 복잡한 2D 레이아웃, 전체 페이지 구조
**Flexbox 사용**: 네비게이션 바, 카드 내부, 버튼 정렬

둘 다 익혀두면 거의 모든 레이아웃 문제를 해결할 수 있습니다.
