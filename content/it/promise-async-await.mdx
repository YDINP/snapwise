<!-- step:cinematic-hook -->
콜백이 7단계로 중첩됐다.
코드가 피라미드처럼 쌓여서
오른쪽으로 사라졌다.

<!-- step:scene -->
API 호출 → DB 저장 → 이메일 발송 → 로그 기록
각 단계가 콜백 안에 콜백.
"콜백 지옥"이었다.

<!-- step:dialogue:senior -->
"Promise로 바꿔봐.
체인으로 연결할 수 있어."

<!-- step:narration -->
`.then()`으로 바꿨다.
피라미드가 일직선이 됐다.

<!-- step:dialogue:junior -->
"읽기는 쉬워졌는데,
여전히 체인이 길어요."

<!-- step:dialogue:senior -->
"async/await 써봐.
동기 코드처럼 보이지만 비동기로 작동해."

<!-- step:impact -->
**const user = await fetchUser();
const post = await savePost(user);
await sendEmail(post);**

<!-- step:narration -->
마치 순서대로 실행되는 것처럼 읽힌다.
하지만 뒤에서는 비동기로 돌아간다.

<!-- step:dialogue:senior -->
"에러 처리는 try/catch로.
Promise는 .catch()였지."

<!-- step:dialogue:junior -->
"병렬로 실행하려면?"

<!-- step:dialogue:senior -->
"Promise.all()로 묶어.
두 API를 동시에 호출하면 시간이 절반."

<!-- step:reveal-title -->
⚡ Promise vs async/await 완벽 정리
콜백 지옥 → Promise 체인 → async/await
동기처럼 읽히는 비동기 코드.

<!-- step:outro -->
**가독성 2배 향상.**
async/await + Promise.all로 깔끔하고 빠르게.
