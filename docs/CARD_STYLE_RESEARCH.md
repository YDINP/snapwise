# 카드 연출 스타일 리서치 결과

> 2026-02-20 | MangaSceneStep 대안 스타일 연구

## 1. 키네틱 타이포그래피 (Kinetic Typography) ⭐ 구현중
- **난이도**: Medium
- **컨셉**: 텍스트 자체가 주인공. 글자 크기/위치/회전 변화로 감정과 임팩트 전달
- **참고**: Apple Keynote, 인스타 릴스 자막, Spotify Wrapped
- **장점**: 텍스트 중심 콘텐츠에 최적, 이미지 불필요, 모바일 가독성 우수
- **구현**: Framer Motion spring/stagger, 타입별 타이포 레이아웃 변형

## 2. 글래스모피즘 매거진 (Glassmorphism Magazine)
- **난이도**: Low-Medium
- **컨셉**: 반투명 유리 카드 + 배경 블러 + 그라데이션
- **참고**: iOS 위젯, Windows 11 Mica, Figma 트렌드
- **장점**: 고급스러운 매거진 느낌, CSS backdrop-filter로 구현 간단

## 3. 네오브루탈리즘 플래시카드 (Neobrutalism Flashcard)
- **난이도**: Low
- **컨셉**: 두꺼운 보더(3-4px), 강렬한 색상, 오프셋 박스 섀도우
- **참고**: Gumroad, Figma Community, Linear
- **장점**: 구현 단순, 시각적 임팩트 강함, 트렌디

## 4. 스크롤리텔링 다큐 (Scrollytelling Documentary)
- **난이도**: Medium-High
- **컨셉**: 다큐멘터리 스타일, 시네마틱 전환, 패럴랙스
- **참고**: NYT Snowfall, Bloomberg, The Pudding
- **장점**: 몰입감 최고, 스토리텔링에 최적

## 5. 게이미피케이션 퀘스트 (Gamification Quest)
- **난이도**: Medium
- **컨셉**: RPG 퀘스트 UI, 진행바, 경험치/레벨 느낌
- **참고**: Duolingo, Forest, Habitica
- **장점**: 재미 요소, 진행감, 완료 동기부여

## 6. 미니멀 카드덱 (Minimal Card Deck / Notion-style)
- **난이도**: Low
- **컨셉**: Notion/Apple 스타일, 깔끔한 여백 활용, 타이포 중심
- **참고**: Notion, Apple HIG, Linear
- **장점**: 가독성 최고, 콘텐츠 집중, 구현 간단

## 7. 인터랙티브 인포그래픽 (Interactive Infographic)
- **난이도**: High
- **컨셉**: 데이터 시각화 중심, 차트/그래프 애니메이션
- **참고**: Information is Beautiful, D3.js 예시
- **장점**: 데이터 패널에 특화, 시각적 정보 전달력 최고
